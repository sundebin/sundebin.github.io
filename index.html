独当一面的前端工程师

技术能力

  • 职业技能
    • 熟练掌握 HTML、CSS、JavaScript
      • 用语义化的 HTML 标签配合 CSS 百分百还原设计稿，并使用 JavaScript 完成各种交互逻辑。
    • 熟练掌握一门 MVVM 框架
      • 了解框架出现的意义，框架的职责边界，你能轻松应用框架提供的各种能力辅助开发工作，学习框架源码，了解框架的实现原理。
    • 至少要学会一种打包工具
      • 以 webpack 为例，你需要熟练掌握 webpack 的配置，学会利用 webpack 辅助我们的开发、部署和优化。
    • 熟练掌握前端的一些广度知识
      • HTTP、Web 安全、性能优化、数据结构和算法等。
  • 开发效率
    • 工作中善于提炼可复用的代码和逻辑
    • 善于通过技术的手段去提升效率
  • 解决问题
    • 线上问题解决研究
    • 探究问题的本质
    • 学会提问
  • 代码重构
    • 不要过度设计
    • 及时重构
    • codereview
    • 学习优秀源码
  • 技术沉淀
    • 技术分享
    • 开源
  • 快速学习
    • 阅读官方文档
    • 工作实践
  • 技术选型
  • 造轮子

业务能力

  • owner 意识
  • 产品思维
  • 沟通能力
  • 敬畏质量及结果
